Historical Libraries Project

Source previews, catalogue materials, and experimental tools for studying early modern intellectual history.
Claim ceiling. This site presents source texts, computational routing evidence, heatmaps, and review metadata to support scholarly inspection. It does not claim that Hume read, used, borrowed from, or was influenced by any listed text unless a separate human-scholar claim is explicitly stated and documented.

This site brings together working materials from several related humanities projects: the La Flèche / Riboulet library catalogue, English-language comparison texts, and a developing Founding Fathers track. It is designed as a scholar-facing workspace for browsing source previews, catalogue materials, and experimental research interfaces.

What this project has built

A public scholarly application around the Collège de La Flèche / Riboulet library catalogue — the library environment where David Hume worked in the 1730s. These are public-safe project surfaces (source previews, catalogue materials, and experimental tools), not proof of Humean influence, contact, or source-dependence.
