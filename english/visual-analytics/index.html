Retrieval matrix — experimental review visualization

Experimental review visualization · for scholarly inspection. This is an experimental, in-progress surface built from the project's committed retrieval metadata. It is deliberately visible while it is still being improved. It shows routing / review structure — how the retrieval channels behave for each candidate relation — not a finding.

This is an experimental review visualization for scholarly inspection. Every layer shows retrieval / routing metadata within a defined scope. It does NOT establish that Hume read, used, borrowed from, contacted, depended on, or was influenced by any comparison text.

How to read this

The grid is shown once per layer. Rows are Hume loci (grouped by Treatise book / the Enquiry); columns are the nine English comparison texts; each cell is the relation record(s) at that intersection and links to the relation page. The layers are kept separate, each on its own scale — the categorical routing layer is never mixed with the raw channels, and no two layers share a colour scale. A single fused "strength" score is deliberately not produced.

  • Categorical routing — the review category (top-surface / near-surface / … ). Category, not strength.
  • Dense cosine — semantic-embedding similarity (multilingual-e5-large). Values sit in a compressed-high band (~0.82–0.895); the colour encodes position within that band, not a 0–1 scale.
  • Local IDF and Char-3gram Jaccard — lexical / orthographic channels (shared weighted vocabulary; character overlap). Sensitive to cognates and OCR.
  • RRF fused rank and Bounded rank — fusion shown as rank / order only; the RRF magnitude is never shown.
  • Disagreement + magnet overlay — the three channel ranks side by side, with cells outlined where a lexical channel ranks a pair far better than the dense channel. A magnet is a review hazard to scrutinise, not a positive signal.

Three worked examples

ENG-v02-023 · Hutcheson × THN 3.2.1
Lexical / Jaccard-heavy — dense-rejected (magnet-suspect)
dense #10698 · IDF #2009 · Jaccard #8 · RRF #44 · bounded ◆#44 · spread 10690
The character-overlap channel ranks it near the top while the dense channel rejects it — a likely orthographic/cognate coincidence. On the categorical map it reads as 'top-surface'; the channel split reveals it as a magnet to scrutinise, not to feature.
ENG-v02-019 · Hobbes × THN 2.3.9
Dense-driven — very high cross-channel spread
dense #1 · IDF #19 · Jaccard #137670 · RRF #1 · bounded ◆#1 · spread 137669
Dense ranks it near the top while Jaccard sits near pool-bottom — the opposite of a magnet. The disagreement is informative (dense-supported, surface-divergent) and is kept visible, not blended away.
ENG-v02-004 · Berkeley Dialogues × THN 1.4.5
All-channel agreement (contrast case)
dense #4 · IDF #11 · Jaccard #17 · RRF #1 · bounded ◆#1 · spread 13
Dense, IDF, and Jaccard all rank the pair closely (small spread) — the well-behaved case that makes the two atypical patterns above legible by contrast.

RRF and bounded values are ranks (order within a defined pool), never magnitudes. Every raw layer below carries its own six-label header (metric · channel · model+frozen-version+run · denominator+cap-state · scope/direction · routing≠evidence).
