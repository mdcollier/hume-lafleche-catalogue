British Contexts is being developed in collaboration with Amyas Merivale, whose English Philosophical Texts Online files provide structured text and paragraph-level metadata for citation, alignment, and review. The full source texts are not republished here.

Visual review tools

Review signal map
A categorical routing overview — where each candidate relation sits by review category (top-surface, near-surface, and so on). Shows routing structure, not strength.
Open the signal map →
Retrieval matrix experimental
Experimental multi-channel visual analytics — an integrated matrix of the dense, IDF, and Jaccard retrieval channels with filters and a per-cell detail panel. Channels are shown separately, never fused into one score.
Open the retrieval matrix →

These are review/routing tools for scholarly inspection. They do not establish that Hume read, used, borrowed from, contacted, depended on, or was influenced by any comparison text.
