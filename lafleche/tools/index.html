Catalogue & comparison tools

The project's original library-first interface for the Riboulet catalogue and its Hume-comparison layers.

These tools orient you in the wider Riboulet catalogue and index the project's metadata-only Hume × text comparison zones. They do not display source texts, and they identify material for scholarly inspection rather than establishing influence, contact, or source-dependence.

Review zones
The main comparison index: metadata-only Hume × text pairings for scholarly inspection. Structural comparison, not influence; no source text.
Catalogue coverage map
A section-level orientation map showing which parts of the Riboulet catalogue are represented in current records and which remain unprocessed. Not an influence map; not a full-catalogue coverage claim.
Original scholar interface
The full library-first entry point for the catalogue and its comparison layers, with the project's own navigation.

A separate catalogue-browser view is still incomplete and is kept in the archive. These interfaces predate the current source-preview release and carry their own older navigation.
