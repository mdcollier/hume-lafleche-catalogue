Experimental humanities tools

Experimental metadata and review tools for orienting yourself in the Riboulet catalogue.

These tools help you orient in the wider Riboulet catalogue — 22 sections and roughly 2,493 entries — and index the project's metadata-only comparison/review records. They do not display source texts, and they identify material for scholarly inspection rather than establishing influence, contact, or source-dependence. To browse the catalogue itself, use the current catalogue sections & folders page.

Review zones
Experimental comparison/review metadata: metadata-only Hume × text pairings for scholarly inspection. Structural comparison, not influence; no source text.
Catalogue coverage map
Section-level representation/orientation: all 22 sections with per-section entry and record counts, showing which parts are represented (6) and which remain unsearched (16). Not an influence map; not a full-catalogue coverage claim.

Older dashboard

Older navigation surface retained for URL continuity. Most readers should use the current catalogue page, the coverage map, or the review zones instead. — Original dashboard (superseded navigation surface).
