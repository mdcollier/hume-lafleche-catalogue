Claim ceiling. This site presents approved source previews, catalogue materials, computational routing metadata, and review records to support scholarly inspection. It does not claim that Hume read, used, borrowed from, contacted, depended on, or was influenced by any listed text.

This is the project's experimental / research-infrastructure dashboard. It gathers the public-safe tools and documentation built so far — experimental studies and method notes, report and method records, and review-workflow interfaces — and previews the analytical surfaces now in preparation.

Everything here is metadata-only and does not display source texts or establish Humean influence, contact, source-dependence, reading history, discovery, validation, recall, or coverage.

What you are looking at

The app holds four kinds of artifacts, each with a different evidentiary status:

Catalogue records
Bibliographic orientation over the Riboulet catalogue — availability and classification context. Not evidence that Hume read or used a text.
Approved source previews
Readable public working texts, shown only when separately cleared — four approved. Normalized working text, not critical editions.
Method / report records
Public-safe summaries of experiments, method checks, validation, and governance — how the instrument works. Not source-dependence, influence, contact, or reading-history claims.
Review-zone metadata
Metadata-only routing / review surfaces — not public cards, not confirmed candidates. No passages, scores, ranks, pair tables, heatmaps, or dossiers.

How review works

The project is a candidate-surfacing instrument: it orders comparison material into review queues so scholars can inspect it. Each rung below is a step in that review workflow, not a finding.

  1. Catalogue record — a bibliographic entry in the Riboulet catalogue.
  2. Processed / routed record — the frozen (version-locked) method has surfaced comparison material for review — routing metadata, not evidence.
  3. Review-zone metadata — a metadata-only comparison / review record for scholarly inspection — not a public card.
  4. Method / report record — a public-safe summary of how the work was done — not a source-dependence claim.
  5. PI / human review — qualified scholarly judgement decides what is worth escalating.
  6. Public source preview — the readable working text, shown only when separately approved.
  • Processing and routing surface candidates for review; they are not evidence of influence.
  • Review zones are metadata for inspection, not public cards or confirmed contact.
  • Method and report records describe how the work was done, not source dependence.
  • The analytical surfaces below (comparison-signal distribution, review-priority tables) remain review-gated and are not published.
  • The claim ceiling above applies to everything on this page.
