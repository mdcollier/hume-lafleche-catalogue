Experimental humanities tools

The project's original library-first interface for navigating the Riboulet catalogue and its review layers.

These tools orient you in the wider Riboulet catalogue — 22 sections and roughly 2,493 entries — and index the project's metadata-only Hume × text comparison zones. They do not display source texts, and they identify material for scholarly inspection rather than establishing influence, contact, or source-dependence. For a static overview of the catalogue's sections and coverage, see the Riboulet catalogue — sections & coverage.

Review zones
The main comparison index: metadata-only Hume × text pairings for scholarly inspection. Structural comparison, not influence; no source text.
Catalogue coverage map
The interactive section-level map: all 22 sections with per-section entry and record counts, showing which parts are represented (6 sections) and which remain unsearched (16). Not an influence map; not a full-catalogue coverage claim.
Original scholar interface
The full library-first entry point for the catalogue and its comparison layers, with the project's own navigation.

A separate catalogue-browser view is still incomplete and is kept in the archive. These interfaces predate the current source-preview release and carry their own older navigation.
